PROJECTS

2021-2022

Underground Cable Fault Detection Using Arduino Uno

This Project proposes fault location model for underground power cable using microcontroller. The aim of this project is to determine the fault in underground cable. When any fault like open circuit occurs, voltage drop varies since the current varies. A set of relays are used for switching device for the particular faults. The main objective of our project is to modernize the Indian power distribution sector aiming to minimize the power loss which can be achieved by using “Arduino Based Underground Cable Fault Detection”.
